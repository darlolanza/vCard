# Dario Lanza - Modern vCard (2026 Design)

This project uses **HTML5** and **Tailwind CSS** to create a futuristic, glassmorphism-styled profile page.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server** (Watches for changes):
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

-   `index.html`: Main content file.
-   `input.css`: Source CSS file with Tailwind directives.
-   `tailwind.config.js`: Configuration for colors, fonts, and animations.
-   `output.css`: Generated CSS file (created after running build/dev command).
